# minesweeper-client
Client for Minesweeper-API

## Use
To get help for each command use `python <command>.py --help`

### Create
`python create.py --height=2 --width=3 --mines=1`

### Update
`python update.py --id=1 --x=0 --y=0 --command=uncover`

### Index
`python index.py`

### Show
`python show.py --id=1`
